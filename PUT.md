# PUT

## PUT a Project

**URL** : `/api/v1/projects/1`

***Method*** : `PUT`

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Content example**
```json
{
    "id": 1,
    "project
}
```
## PUT a specific palette


**URL** : `/api/v1/projects/:id`

***Method*** : `PUT`

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Content examples**

For a project with an ID of 1:`/api/v1/projects/1`

```json
{
    "id": 1
}
```
