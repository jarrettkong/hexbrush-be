# Show Current User

Get the details of the currently Authenticated User along with basic
subscription information.

**URL** : `/api/v1/projects/`

***Method*** : `GET`

***Auth required*** : YES

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Content examples**

For a project with an ID of 2:
```json
{
    "id": 1,
    "name": "Idea Box",
}
```

For a project with an ID of 2:

```json
{
    "id": 2,
    "first_name": "",
  
}
```
